== Installation ==

= Plugin Installation =
1. Upload the plugin folder to the "/wp-content/plugins/" directory of your WordPress site,
2. Activate the plugin through the 'Plugins' menu in WordPress,
3. Visit the "Facebook and Twitter Login" administration page to setup the plugin.

= API Connection =
The social network APIs are constantly changing and being updated. We monitor these changes and automatically
update our APIs, so that you can be sure that Facebook and Twitter Login will always run smoothly and with the most up-to-date
API calls.


== Frequently Asked Questions ==

= Do I have to add template tags to my theme? =

You should not have to change your templates.
The Facebook and Twitter Login seamlessly integrates into your blog by using predefined hooks.

= Does Facebook and Twitter Login can be embedded through a shortcode? =

We created seperate shortcode for facebook and twitter for convenience. Facebook and Twitter Login shortcodes are `[fb_user_login]` for facebook
and `[tw_user_login]` for twitter can be used in any page or post within your blog.
The shortcode will automatically be replaced by the icons of the chosen
social network providers.

Do not hesitate to contact us at www.coralwebdesigns.com if you need further assistance.

= My users cannot login with facebook login

If you encounter any problems with facebook login , please consider checking the callback url to fix the problem.

= Do I have to option to add Facebook to Twitter login to my wordpress login page? =

Yes, you have option to add Facebook and Twitter login to wordpress login page.
